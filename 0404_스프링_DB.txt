1. root-context.xml 파일 이동
 - src/main/resources 폴더에 config.spring.context 패키지 생성
 - root-context.xml 파일 이동
 - 이름 변경 --> context-1-datasource.xml
 - 복사로 파일 하나 더 생성 --> context-2-mybatis.xml
 - 복사로 파일 하나 더 생성 --> context-3-dao.xml
2. web.xml에서 root-context.xml의 참조 정보 변경
 - <param-value>classpath*:config/spring/context/context-*.xml</param-value>로 변경
 - classpath*: --> 클래스 패스에 있는 모든 경로를 탐색
3. servlet-context.xml 파일 이동
 - src/main/resources 폴더에 config.spring.context 패키지 생성
 - servlet-context.xml 파일 이동
 - 경로를 바꿔주어야 한다.
 - <param-value>classpath*:config/spring/mvc/servlet-context.xml</param-value>
4. mybatis 세팅
 - src/main/resources 폴더에 config.mybatis 패키지 생성
 - db.properties, mybatis-config.xml 파일 이동
 - db.properties에 db정보 수정
 - mappers 태그에 mapper 태그로 참조할 mapper 설정
5. mapper 세팅
 - src/main/resources 폴더에 config.mybatis.mapper 패키지 생성
6. mvnrepository.com에서 라이브러리 세팅
 - 라이브러리끼리 충돌이 잘 되므로 버전을 적당히 선택해야 한다.
 - commos-dbcp 1.4 버젼 다운
 - mybatis 3.4.5 버젼 다운
 - mybatis-spring 1.3.1 버젼 다운
 - Spring JDBC 3.2.2 버젼 다운
7. context-1-datasource.xml 설정
 - db.properties와 연결
 - 하위 탭 Namespaces에서 context 체크
 - <context:property-placeholder location="classpath:config/mybatis/db.properties"/>
